## Prompts 
Prompts utilizados baseados nos principais tópicos abordados na aula e nas avaliações da faculdade
- Qual a diferença entre o operário do Taylorismo e do Fordismo?
- Quais são as principais críticas e limitações desse modelo administrativo?
- Como o conceito de 'homo economicus' influenciava a motivação dos operários?
- Como a Escola das Relações Humanas propôs motivar o operário?
